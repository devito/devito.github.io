This is a demo for loading resources with html elments

Look in console

Flow is the following:

  • add iife_script as module script tag script src="/script/iife_script.js"
  • script has a global function with a global counter that increments per load
  • call function with regular inline script tag
  • load iife_script again with scriptsrc="/script/iife_script.js"
  • call function with module regular script tag one last time
  • in console will see that script is loaded twice and state is updated
